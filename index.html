NorthStar is an app I made to help myself focus on one task at a time. NorthStar is a recursive to-do list, and when a task is pinned, it will remain on the screen at all times, keeping you on track. Available on the Mac App Store: Mac App Store - NorthStar

A side project where I trained a neural network to trade stocks using the NEAT Python library. It scrapes live stock data and interacts with a simulated trading environment I made to learn buy/sell behavior over time. Did it work? Not yet, but I continue working on it.
